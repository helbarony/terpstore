# 🐢 Terp Store

The **Terp Store** is a web-based application that allows users to browse store items, inquire about products, and submit orders online.  
It provides a simple interface for viewing available merchandise and placing orders through user-friendly forms.

---

## ✨ Features
- 🛍️ Browse and inquire about store items  
- 🧾 Submit product orders easily through forms  
- 📄 Two separate forms for inquiries and order submissions  
- ⚡ Fast and lightweight backend setup  
- 🎨 Clean, user-friendly interface
